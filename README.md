# Taccuino

A lightweight, client-side note-taking app built with React 19, TypeScript, Vite, Tailwind CSS v4, and shadcn/ui.

## Features

- Create, edit, and delete notes
- Pin important notes to the top
- Tag notes for organization
- Search and filter notes by title, content, or tags
- Full-text search across all notes
- Markdown rendering for note content
- Dark mode support
- Export and import notes as JSON
- Automatic persistence via localStorage

## Stack

- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite 7
- **Styling:** Tailwind CSS v4
- **UI Library:** shadcn/ui (New York style)
- **Icons:** lucide-react

## Getting Started

```bash
npm install
npm run dev
```
